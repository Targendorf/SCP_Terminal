# SCP Terminal — CRT 1991

Интерактивный имитатор компьютерного терминала Фонда SCP для настольных ролевых игр. Игроки вводят пароль и получают доступ к папкам и файлам разных терминалов; мастер может создавать/редактировать терминалы через админ-панель.

## Пароли из seed-данных (для примера)

| Пароль          | Доступ                                |
|-----------------|---------------------------------------|
| `valkyrie`      | Site-19 Reception (Level 1)           |
| `keter-rising`  | Site-19 Containment Archive (Level 3) |
| `oracle-five`   | O5 Command (Level 5)                  |
| `overlord-13`   | **Мастер-пароль** → админ-панель      |

## Деплой на Vercel

Проект статический — достаточно подключить репозиторий к Vercel. Никакой серверной логики не нужно.

### Как работает сохранение

- **Seed-канон** (`data/seed.js`) — фиксированный набор терминалов в репозитории. Это то, что игроки видят при первом заходе и после "сброса".
- **localStorage** — все правки мастера (создание терминалов, редактирование файлов) автоматически сохраняются в браузере. У каждого устройства своя копия.
- **Экспорт JSON** — мастер может скачать всю свою вселенную одним JSON-файлом и отдать другому мастеру или сохранить между устройствами.
- **Импорт JSON** — загрузить ранее сохранённый JSON и заменить локальное состояние.
- **Чтобы зафиксировать изменения в "канон"** — экспортируйте JSON и подложите его содержимое в `data/seed.js` как значение `window.SCP_SEED`.

## Управление

- `↑` `↓` — выбор пункта
- `Enter` — открыть
- `Esc` — назад / выйти
- `\` или `` ` `` — командный режим (`ls`, `cd`, `cat`, `help`, `logout`, …)

## Tweaks (кнопка в тулбаре)

Язык (RU/EN), интенсивность скан-линий, свечения, шума, вибрации, цвет фосфора (зелёный / янтарный / белый), звук, гул CRT.
